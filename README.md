# Projekt systemów wbudowanych

Projekt z zakresu systemów wbudowanych obejmujący implementację oprogramowania sterującego oraz komunikację pomiędzy komponentami systemu.

W ramach projektu wykorzystano język C oraz środowisko Linux. Realizacja obejmowała rozwój kolejnych wersji oprogramowania, testowanie działania systemu oraz pracę z komunikacją sieciową.

## Technologie

- C
- Linux
- Programowanie systemowe
- Komunikacja sieciowa
- Systemy wbudowane
